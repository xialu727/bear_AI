# 贪吃蛇游戏

这是一个使用 Python 和 Pygame 库开发的经典贪吃蛇游戏。

## 游戏功能

- 经典贪吃蛇游戏玩法
- 开始菜单界面
- 分数和长度显示
- 游戏暂停功能
- 游戏结束界面
- 重新开始和退出选项

## 运行要求

- Python 3.x
- Pygame 库

## 安装和运行

1. 确保已安装 Python 3.x
2. 安装 Pygame 库：
   ```
   pip install pygame
   ```
3. 运行游戏：
   ```
   python test2.py
   ```

## 游戏操作

### 开始界面
- 按 **空格键** 开始游戏

### 游戏控制
- **方向键** 或 **WASD** 控制蛇的移动方向：
  - 上/↑ 或 W：向上移动
  - 下/↓ 或 S：向下移动
  - 左/← 或 A：向左移动
  - 右/→ 或 D：向右移动
- **P键**：暂停/继续游戏

### 游戏结束界面
- **R键**：重新开始游戏
- **Q键**：退出游戏

## 游戏规则

1. 控制蛇吃掉红色食物
2. 每吃到一个食物，蛇的长度增加，得分增加10分
3. 避免蛇撞到自己的身体
4. 蛇可以穿过屏幕边界（从一边出来从另一边进入）

## 游戏特点

- 蛇头用蓝色表示，身体用绿色表示
- 蛇头有眼睛，会根据移动方向变化
- 食物设计为苹果样式，带有褐色果梗
- 游戏界面有网格背景
- 实时显示得分和蛇的长度

## 文件说明

- `test2.py`：游戏主程序文件
- `README.md`：本说明文档

## 注意事项

- 游戏默认速度为每秒10帧
- 菜单和结束画面使用每秒30帧
- 蛇不能直接反向移动（例如不能从向右直接变为向左）
